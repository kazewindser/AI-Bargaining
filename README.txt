
# How to run

1) Install oTree if you haven't:
   pip install otree>=5.10

2) In a terminal:
   cd myproj_ai_bargaining
   otree devserver

3) Open http://localhost:8000 in your browser.
   - Click "ai_bargaining_demo" to create a session.
   - Open the 2 participant links (e.g., in two different browser windows) to test P1/P2.
